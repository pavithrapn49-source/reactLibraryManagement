# React Library Management

A React-based application for managing library resources, including books, users, and borrowing operations.

## Features

- Add, edit, and delete books from the library
- Manage library users and their profiles
- Track book borrowing and returns
- Search and filter library inventory
- View borrowing history

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Available Scripts

- `npm start` - Start the development server
- `npm test` - Run tests
- `npm run build` - Build the app for production
- `npm run eject` - Eject from Create React App

## Technologies Used

- React
- React Router
- CSS/Styled Components
- JavaScript ES6+

## Project Structure

```
src/
├── components/     - React components
├── pages/         - Page components
├── services/      - API services
├── styles/        - CSS files
└── App.js         - Main App component
```
